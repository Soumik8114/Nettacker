OWASP Nettacker

Automated Penetration Testing & Security Scanning Framework

OWASP Nettacker is a comprehensive security framework designed to automate information gathering, vulnerability scanning, and network assessments. It helps security professionals identify and document weaknesses across networks, web applications, IoT devices, and APIs.

Getting Started:
  • Click New Scan to create your first security assessment
  • Use Crawler to view scan results organized by target
  • Check Results to review past scans
  • Visit the Tutorial for a guided walkthrough

Key Features

  • Multi-protocol scanning
  • Port & service detection
  • Vulnerability assessment
  • Credential brute-forcing
  • Custom reporting
  • Scan history & comparison
